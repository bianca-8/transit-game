### Transit Game

This is a hub for a lot of games to play with your travel buddy while you're waiting to get to your destination. I want them to be multiplayer, and focused on building a stronger relationship with the a person you're travelling with. The different games buttons based off of places I've travelled to in the past, and the photos are actual photos from my trips! If you have any suggestions for games let me know!

I was kind of inspired by the Messages games with a friend, specifically the 20 questions one, since I was playing it on a 1hr bus trip that I frequently take. I was actually learning a lot about this person while I was playing the game with them, and I want to make a hub for multiplayer games designed for you to learn more about the person you are playing with to become better friends :D

Right now, the map and all the buttons work, so I just need to work on the actual games and design of the whole thing after.
